Exemplo de Documento para RAG

Este é um documento de exemplo para demonstrar o funcionamento da ferramenta RAG.

Sobre Inteligência Artificial:
A Inteligência Artificial (IA) é um campo da ciência da computação dedicado a criar sistemas capazes de realizar tarefas que normalmente requerem inteligência humana.

Principais Conceitos:
1. Machine Learning: Algoritmos que aprendem com dados
2. Deep Learning: Redes neurais profundas
3. Natural Language Processing: Processamento de linguagem natural
4. Computer Vision: Visão computacional

Aplicações Práticas:
- Assistentes virtuais
- Sistemas de recomendação
- Diagnóstico médico
- Veículos autônomos
- Análise de sentimentos

Agentes de IA:
Agentes são sistemas autônomos que percebem o ambiente e tomam ações para atingir objetivos específicos. Podem ser simples ou complexos, trabalhando individualmente ou em equipes (multi-agentes).
